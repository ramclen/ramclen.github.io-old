SIANI End of Degree Project

Simulation Gateway Interface o SGI, representa de forma sencilla simulaciones de sistemas complejos. Para ello se ofrece un Framework, con el que el desarrollador puede adaptar fácilmente las simulaciones, y un cliente web, el cual nos permite visualizar y controlar la ejecución de la simulación.

SGI se unió a un proyecto existente en el laboratorio del SIANI, llamado TAFAT.
